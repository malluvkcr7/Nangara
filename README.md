## Get started

1. Get your emulator or simulator running
2. Update the .env file with your Groq api key.
3. Update your firebase config details in the src/services/firebase.js
4. Install dependencies

   ```bash
   npm install
   ```
5. Start the app

   ```bash
    npx expo start
   ```
